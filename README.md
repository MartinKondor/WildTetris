# WildTetris
[![version](https://img.shields.io/badge/version-prototype-red.svg)](https://github.com/MartinKondor/WildTetris)

WildTetris is a simple tetris game built in C++.

## Usage

Build 

```shell
make build
```

Run 

```shell
./bin/wildtetris.exe
```

## Structure

```
src/  # headers and source files
tests/  # TDD tests
```
